# Congruity-3.0-Lite

Versione pubblica e divulgativa della metrica ICᵀ  
(*Indice di Congruità Totale*) – proporzionalità reale nei sistemi complessi.

## Descrizione
Questo repository presenta una versione **semplificata e statica (Lite)** del framework Congruity,
finalizzata a scopi **educativi, esplorativi e dimostrativi**.

La metrica ICᵀ valuta la coerenza proporzionale tra:
- Valore generato
- Costo
- Distanza
- Energia
- Complessità

in contesti economici, logistici, tecnologici e decisionali.

## Ambito di utilizzo
Questa versione **NON include**:
- pesi adattivi o dinamici
- soglie decisionali
- criteri di esclusione
- modelli temporali o predittivi
- sistemi di governance o controllo decisionale

È pensata esclusivamente per:
- studio concettuale
- sperimentazione individuale
- uso non commerciale

## Proprietà intellettuale
L’architettura completa del framework ICᵀ, le versioni avanzate (≥ 7.0),
i meccanismi di decision-level constraints e le estensioni AI/AGI
sono **proprietà intellettuale dell’autore** e **non sono rilasciate** in questo repository.
