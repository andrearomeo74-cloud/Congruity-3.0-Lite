# Congruity-3.0-Lite

Public and educational version of the ICT (Total Congruity Index) — real proportionality in complex systems.

---

## Description

This repository presents a **simplified and static (Lite)** version of the Congruity framework, designed for **educational, exploratory, and demonstrative purposes**.

The ICT metric evaluates proportional coherence between:

- Generated value  
- Cost  
- Distance  
- Energy  
- Complexity  

across economic, logistical, technological, and decision-making contexts.

---

## Scope of Use

This version does **NOT** include:

- adaptive or dynamic weights  
- decision thresholds  
- exclusion criteria  
- temporal or predictive models  
- governance or decision-control systems  

It is intended **exclusively** for:

- conceptual study  
- individual experimentation  
- non-commercial use
